Answers to the following questions in a plain text file

1. Name one advantage to use GitHub and one advantage to using the CPanel hosting (these need to be different)?
One advantage of GitHub is that users don't need to download an extra FTP app to host a website.
One advantage of Cpanel is that it is simple to navigate because it usually comes with an icon-based menu.


2. Which type of hosting do you prefer and why?
I prefer GitHub because after the user links the repository to a local file, it will remind the user to fetch all the changes in GitHub Desktop.
